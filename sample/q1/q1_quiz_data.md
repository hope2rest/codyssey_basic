## 문제

`data/quiz_data.csv` 파일에 저장된 퀴즈 데이터를 읽어와 채점하는 프로그램을 작성하세요.

`q1_quiz_data.py`의 `main()` 함수를 완성하여 결과 딕셔너리를 반환하세요.

---

## 데이터 형식

`quiz_data.csv`는 다음 컬럼을 가집니다:
- `id`: 문항 번호 (정수)
- `question`: 질문 텍스트
- `option_a`, `option_b`, `option_c`: 보기 A, B, C
- `answer`: 정답 (A, B, C 중 하나)

---

## 요구사항

### 1. 퀴즈 데이터 로드
- `csv` 모듈을 사용하여 `data/quiz_data.csv`를 읽어옵니다.
- 각 행을 딕셔너리로 변환하여 리스트에 저장합니다.
- 각 딕셔너리는 다음 키를 가집니다: `id`(int), `question`(str), `options`(dict), `answer`(str)
- `options`는 `{'A': '보기A', 'B': '보기B', 'C': '보기C'}` 형태입니다.

### 2. 채점
- 사용자 답안 리스트 `['B', 'A', 'B', 'C', 'A']`와 정답을 비교합니다.
- 각 문항별로 정답 여부를 판별합니다.

### 3. 결과 반환
- `main()` 함수에서 아래 형식의 딕셔너리를 반환합니다.

---

## 출력 형식

```python
{
    "total": 정수,          # 총 문항 수
    "correct": 정수,        # 맞힌 문항 수
    "score": 실수,          # 점수 (백분율, 소수점 1자리)
    "details": [            # 문항별 상세 결과
        {
            "id": 정수,
            "correct_answer": "문자열",
            "user_answer": "문자열",
            "is_correct": bool
        },
        ...
    ]
}
```

---

## 채점 기준 (4항목, 각 25점)

| 항목 | 검증 내용 |
|------|----------|
| total | 총 문항 수 정확 일치 |
| correct | 맞힌 개수 정확 일치 |
| score | 점수 정확 일치 |
| details | 상세 결과 구조 및 값 검증 |
